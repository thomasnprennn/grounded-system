# Raw Input

Use this folder to store all source material from the client before any diagnosis is written.

Rules:

- Keep raw evidence unchanged.
- Do not summarize evidence here unless needed for file labels.
- Do not claim analysis until the relevant evidence exists.
- Put screenshots in `/screenshots`.
- Put caption text or exports in `/captions`.
- Put owner/client notes in `client-notes.md`.
- Put customer comments, DMs, review excerpts, or repeated questions in `customer-comments.md`.
