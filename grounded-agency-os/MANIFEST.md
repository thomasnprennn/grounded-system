# File Manifest

This repository contains the first complete structure for `gr( )unded studio` agency operating system.

## Root

- `AGENTS.md` — main instruction file for Codex/AI agents.
- `README.md` — how to use this repository.
- `MANIFEST.md` — list of files and what they are for.
- `NEEDED-DECISIONS.md` — questions still needed to make the system final.

## Brand

- `brand/brand-guide.md` — master brand guide.
- `brand/identity.md` — name, tagline, mission, vision, values, positioning.
- `brand/tone-of-voice.md` — writing rules, words to use/avoid, platform tone.
- `brand/visual-system.md` — logo, colors, typography, visual world.
- `brand/offer-positioning.md` — service package positioning and decision logic.

## Operating System

- `operating-system/ai-human-boundary.md`
