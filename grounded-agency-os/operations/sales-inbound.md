# Sales Inbound Flow

## 01 — First Response

Goal: understand the problem, not sell immediately.

Message:

Halo [nama]. Boleh cerita dulu — brand kamu sekarang ada di fase apa, dan masalah yang paling ingin diselesaikan apa?

## 02 — Diagnose Need

Listen for:

- unclear problem → Foundation Audit
- system problem → Brand System Build
- execution need → Full Implementation
- specific landing page need → Landing Page workflow

## 03 — Explain Package by Problem

Do not list all packages.

Say:

"Dari konteks ini, yang paling tepat adalah [package], karena [problem]."

## 04 — Move to Intake / Call

Audit:
- send intake form
- invoice DP

Build / Full:
- set discovery call
- scope mapping
- proposal

## 05 — Record

Save conversation summary to:

`/clients/[client]/messages/inbound-summary.md`
