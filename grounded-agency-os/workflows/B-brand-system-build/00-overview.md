# B — Brand System Build Overview

Package: Brand System Build
Price: Rp 7.500.000
Timeline: 4–6 weeks
Role: hero product / main delivery

## Objective

Build a complete clarity system for a brand: strategy, visual language, brand guide, IG visual world, and website direction.

## Workflow

1. B1 — Onboarding + Data Collection
2. B2 — Brand Discovery
3. B3 — Strategic Foundation
4. B4 — Visual System Direction
5. B5 — Brand Guide Creation
6. B6 — IG Visual World
7. B7 — Website Direction
8. B8 — Client Review + Refinement
9. B9 — Final Delivery + Training

## Rule

No visual direction before strategic foundation is approved.
