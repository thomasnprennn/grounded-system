# D — Landing Page Workflow Overview

This workflow is for landing pages as a focused deliverable or part of Full Implementation.

## Objective

Turn brand strategy and offer clarity into a page that helps customer decision-making.

## Workflow

1. D1 — Offer Diagnosis
2. D2 — Sitemap / Page Structure
3. D3 — Copywriting
4. D4 — Wireframe Direction
5. D5 — Build Brief
6. D6 — QA + Launch Checklist

## Rule

A landing page is not decoration. It is a decision-making system.
