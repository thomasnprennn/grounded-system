# A — Foundation Audit Overview

Package: Foundation Audit
Price: Rp 2.500.000
Timeline: 1 week
Role: entry point + proof-building

## Objective

Diagnose why a brand does not look as credible, coherent, or intentional as its product/service quality.

This is not a design package. This is diagnosis.

## Workflow

1. A1 — Qualify
2. A2 — Intake
3. A3 — Visual Audit
4. A4 — Narrative Audit
5. A5 — Customer Perception Audit
6. A6 — System Coherence Audit
7. A7 — Root Cause Synthesis
8. A8 — Diagnosis Report
9. A9 — Delivery Call
10. A10 — Follow-up + Case Study Setup

## Main Deliverables

- client profile
- intake summary
- visual audit score
- narrative audit score
- customer perception gaps
- system coherence map
- 3 root problems
- priority recommendations
- diagnosis report
- delivery call notes
- follow-up message

## Rule

Do not offer design solutions before root causes are locked.
