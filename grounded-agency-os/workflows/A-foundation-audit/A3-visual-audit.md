# A3 — Visual Audit

## Purpose

Audit the brand visual language from provided screenshots/assets.

## Inputs Needed

- logo screenshot/file
- Instagram feed screenshots
- product/service visuals
- website screenshots if available
- packaging/store visuals if available

## Files to Read

- AGENTS.md
- brand/visual-system.md
- clients/[client]/01-intake.md

## Output to Create

- visual clarity score
- logo assessment
- color consistency assessment
- typography assessment
- photography/style coherence
- visual perception impact

## Save To

`/clients/[client]/foundation-audit/03-visual-audit.md`

## Human Review

Human validates whether visual observations match actual screenshots.

## Quality Gate

- cites provided evidence by screenshot/file name
- separates taste from clarity
- avoids redesign solution
- scores each dimension

## AI Instruction

```txt
Analyze the provided visual materials. Score logo, color, typography, photography, and overall coherence from 1–10. For each dimension, write current state, specific observation, perception impact, and missing data. Do not recommend redesign yet.
```
