# Offer Positioning — gr( )unded studio

## Core Offer Logic

`gr( )unded studio` does not sell design files first. It sells clarity systems.

The offer ladder:

1. Foundation Audit — diagnose what is broken.
2. Brand System Build — build the visual/narrative system.
3. Full Implementation — apply the system across touchpoints.

## Package 1 — Foundation Audit

Price: Rp 2.500.000
Timeline: 1 week
Role: entry point and proof-building

Best for:
- brand that feels inconsistent but cannot name the root problem
- client who wants clarity before committing to a bigger rebuild
- brand that needs an objective diagnosis

Output:
- clarity score
- visual audit
- narrative audit
- customer perception insight
- system coherence map
- root cause analysis
- diagnosis report
- live delivery call

Positioning line:

"Satu minggu, kamu tahu persis apa yang perlu dibenahi dan mulai dari mana."

## Package 2 — Brand System Build

Price: Rp 7.500.000
Timeline: 4–6 weeks
Role: hero product / main delivery

Best for:
- brand that is ready to rebuild its system
- client who already understands inconsistency is not a surface problem
- business that needs repeatable visual and narrative rules

Output:
- onboarding + discovery
- strategic foundation
- visual system
- brand guide
- IG visual world
- website direction
- final training

Positioning line:

"Bangun sistem visual brand kamu dari strategi sampai panduan implementasi — bukan satu file desain, tapi framework yang bisa dipakai terus."

## Package 3 — Full Implementation

Price: Rp 12.500.000+
Timeline: 8–12 weeks
Role: complete transformation

Best for:
- brand that needs strategy and execution
- client who needs website, content, packaging, or photo production aligned
- business that wants the system applied across touchpoints

Output can include:
- everything in Brand System Build
- website development
- photography direction/production
- packaging design
- content calendar
- implementation support
- strategy calls

Positioning line:

"Dari sistem brand sampai eksekusi di semua touchpoint — semua bicara dalam satu bahasa."

## Package Decision Logic

IF client cannot describe the problem clearly → Foundation Audit.

IF client wants to test first → Foundation Audit.

IF client knows the problem and can commit 4–6 weeks → Brand System Build.

IF Foundation Audit shows system-level problems → present Brand System Build naturally.

IF client needs website + content + packaging + implementation → Full Implementation.

IF Full Implementation scope is unclear → do scope mapping call before pricing.
