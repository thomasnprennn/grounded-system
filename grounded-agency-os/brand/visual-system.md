# Visual System — gr( )unded studio

## Logo

Primary name:

`gr( )unded studio`

The bracket `( )` is not decoration. It is a negative-space signature.

## Logo Rules

- lowercase only
- no gradient
- no shadow
- no outline
- no vibrant background
- minimum digital size: 120px
- minimum print size: 30mm

## Logo Font

- Main letters: DM Sans 400
- Bracket: DM Sans 200

## Color Palette

| Name | Hex | Usage |
|---|---:|---|
| Maroon | `#643233` | Logo, CTA, accent |
| Deep Maroon | `#3D1E1F` | Dark backgrounds, depth |
| Dusty Rose | `#8C4E50` | Hover, accent, secondary text on dark |
| Amber Warm | `#C4A882` | Warm highlight, small detail |
| Warm White | `#F5F0E8` | Primary light background |
| Cream | `#E8E0D0` | Secondary surface, card, divider |
| Ink | `#2C2C2A` | Body text, dark alternative |

Never use:

- bright colors
- high-saturation colors
- random seasonal colors
- gradients
- pure white as main background
- pure black as main background

## Typography

### Heading

DM Sans 500, slight negative letter spacing.

### Body

DM Sans 300, generous line height.

### Label

DM Sans 500 uppercase, 0.10–0.12em letter spacing.

### Data / Code

DM Mono 400.

### Quote / Editorial Moment

Cormorant Garamond Italic.

## Visual World

One sentence:

Warm brutalism — studio kecil di gedung beton, pencahayaan jendela besar, orang-orang diam tapi punya taste tinggi.

## Mood Keywords

Warm. Structured. Intentional. Restrained. Confident. Minimal. Moody. Sophisticated. Deliberate. Editorial.

## Material Direction

Use tension between:

- hard material: concrete, stainless steel, raw texture, industrial surface
- human warmth: window light, clothing texture, coffee, hands, paper, lived-in objects

## Lighting

- natural light from large window
- dramatic but soft shadow
- shadow as design element
- no ring light
- no flat white studio background

## Composition

- off-center
- intentional negative space
- one focused object
- imperfect but deliberate
- editorial, not stock-photo

## Instagram Grid Rule

9-grid should feel like one editorial spread.

Pattern:

wide shot → object close-up → human/detail → repeat

No two similar subject shots side by side.

## Digital Application

- background: Warm White, Cream, Deep Maroon, or Ink
- border radius: minimal 4–8px
- generous spacing
- no SaaS-generic UI look

## Print Application

- uncoated paper
- natural white or cream
- blind emboss or maroon foil allowed
- no glossy lamination
- minimum margins 10mm
