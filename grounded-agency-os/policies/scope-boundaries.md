# Scope Boundaries

## Foundation Audit Includes

- audit from provided materials
- clarity score
- root cause diagnosis
- diagnosis report
- delivery call

Does not include:
- logo design
- brand guide
- content creation
- website design
- implementation

## Brand System Build Includes

- strategic foundation
- visual system direction
- logo/color/type system
- brand guide
- IG visual world
- website direction
- training/handoff

Does not include by default:
- website development
- photo production
- packaging production
- ongoing content execution

## Full Implementation Includes

Depends on scope.

Can include:
- website
- photo production
- packaging
- content calendar
- implementation support

Must be scoped before pricing.
