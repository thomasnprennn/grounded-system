# Quality Gates

Use this checklist before accepting any output.

## Universal Quality Gate

- [ ] Uses correct name: `gr( )unded studio`
- [ ] Starts from diagnosis before solution
- [ ] Specific to this client/brand
- [ ] No generic branding advice
- [ ] No hype language
- [ ] No unsupported claim
- [ ] Uses customer perspective
- [ ] Has a clear next step
- [ ] Saved in correct folder
- [ ] Does not mix context from another client

## Audit Quality Gate

- [ ] Includes clarity score
- [ ] Separates symptom from root cause
- [ ] Explains business/customer impact
- [ ] Uses provided evidence
- [ ] Marks missing data
- [ ] Gives priority, not a random list

## Strategy Quality Gate

- [ ] One strong direction, not five weak options
- [ ] Defensible positioning
- [ ] Clear customer archetype
- [ ] Specific value proposition
- [ ] Visual direction follows strategy

## Copy Quality Gate

- [ ] Ready to send
- [ ] Not too formal
- [ ] No filler intro
- [ ] Clear action requested
- [ ] Natural Indonesian with precise English terms

## Design Brief Quality Gate

- [ ] Explains why before what
- [ ] Gives concrete specs
- [ ] Avoids trend-based reasoning
- [ ] Connects visual direction to customer perception
