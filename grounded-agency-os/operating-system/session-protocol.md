# Session Protocol

Use this before every serious work session with AI.

## Session Brief Format

```txt
SESSION BRIEF — gr( )unded studio

CLIENT:
DATE:
PHASE:

LAST SESSION:
[what was done before]

TODAY'S GOAL:
[what must be finished today]

FILES TO LOAD:
[list files AI should read]

DECISIONS ALREADY MADE:
[locked decisions]

OUT OF SCOPE TODAY:
[what should not be discussed]

QUALITY CHECK:
[criteria]
```

## Drift Rule

If the work starts moving outside the session goal, stop and say:

"Ini di luar brief sesi. Park dulu atau ubah scope?"

## Done Rule

A session is done when the stated output exists as a file and the next step is clear.

## Example

```txt
CLIENT: Kopi Senja
PHASE: Audit
TODAY'S GOAL: Complete visual audit from provided screenshots and score each dimension.
FILES TO LOAD:
- AGENTS.md
- brand/brand-guide.md
- workflows/A-foundation-audit/A3-visual-audit.md
- clients/kopi-senja/intake.md
- clients/kopi-senja/assets/screenshots/
OUT OF SCOPE TODAY:
- narrative audit
- root cause synthesis
- recommendations
```
