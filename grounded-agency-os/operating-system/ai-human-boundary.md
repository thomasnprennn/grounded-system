# AI / Human Boundary

## AI Can Do

- read instructions
- organize files
- draft audits
- create root-cause maps
- generate client messages
- write report drafts
- create landing page copy
- prepare checklists
- synthesize notes

## Human Must Do

- decide if a client is worth taking
- validate whether observations are true
- judge visual taste
- talk to client
- approve final strategy
- approve final deliverables
- handle negotiation
- decide pricing exceptions

## Review Requirement

Every client-facing output must be reviewed by human before sending.

Mark files as:

- `internal-draft`
- `human-review-needed`
- `client-ready`

## Red Flag

If AI starts sounding too certain without enough evidence, require it to rewrite with assumptions clearly marked.
